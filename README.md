# Escape-Run
Escape City Run Game in Unity 3D

Various third party plugins were used while developing the project. Some of those: <br />
-DOTween <br />
-Cinemachine <br />
-SkyBox Assets <br />
-Some hypercasual low poly city assets <br />

It is under development.
